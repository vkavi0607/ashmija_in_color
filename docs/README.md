# ashmija in color

Premium Custom Murals & Hand-Painted Wall Art

## Overview

ashmija in color is a premium mural studio landing page that connects spaces and brands with South Asia's finest mural artists. We curate, manage, and craft hand-painted murals and custom wall art experiences.

## Features

- **Hero Section** - Immersive introduction with call-to-action buttons
- **Gallery Portfolio** - Showcase of hand-painted mural projects
- **Artist Directory** - Meet the talented team of artists
- **Testimonials** - Client reviews and testimonial cards with glass-morphism design
- **Rate Us Modal** - Interactive review submission with avatar picker
- **Responsive Design** - Fully responsive layout for all devices
- **Botanical Art Branding** - Custom mural artwork integrated throughout
- **Modern Aesthetics** - Glass-card effects, smooth animations, and premium typography

## Tech Stack

- **HTML5** - Semantic markup with accessibility features
- **CSS3** - Flexbox/Grid layout, backdrop filters, animations
- **Vanilla JavaScript** - Intersection Observer for scroll reveals, modal management
- **Fonts** - Playfair Display (headings), Outfit (body), Space Grotesk (accents)

## Color Palette

- **Gold** - `#b8933a` (primary accent)
- **Beige** - `#f7f7f7` (background)
- **Ink** - `#1c1a17` (text)
- **Sage** - `#6b7c6b` (accent)

## Getting Started

1. Clone the repository
2. Open `index.html` in a browser
3. Or run a local server: `python -m http.server 5500`
4. Visit `http://localhost:5500`
